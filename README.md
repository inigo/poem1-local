A local server for the Poem/1 AI clock - https://www.kickstarter.com/projects/genmon/poem-1-the-ai-poetry-clock.

## Overview

I kickstarted the Poem/1 clock - which displays a new LLM-generated poem every minute. 
Since a single repository of poems is being used for every user, the poems displayed
are not necessarily relevant to the local conditions - referencing beams of sunlight 
when there is heavy rain, talking about sunset hours before it actually happens, and 
so on. But - Matt Webb has made it very easy to use your own server instead, so I 
can generate a library of poems using a local open weights LLM, tagged with keywords
like "rainy"; change the Poem/1 to point at it; and pass in relevant tags to the
server based on the current conditions.
